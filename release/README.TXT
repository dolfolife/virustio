----------------------------------------------------------------------------------------------------------------
README for The Virus
----------------------------------------------------------------------------------------------------------------

Requirements:

* A computer (GNU/Linux preferably, but Windows or Mac OS will suffice)
* Python 2.5.2 or greater installed (may run in older versions)
* Pygame 1.8.1 or greater (may run in older versions)

Installation:

* Uncompress the the-virus.zip file in any location.

And... presto!

Running the game:

* In the terminal program, go to the uncompressed folder.

* Run:

$ python virustio.py

And enjoy

(If you use Windows, just double click the virustio.py file in the Windows Explorer).

----------------------------------------------------------------------------------------------------------------
How to play
----------------------------------------------------------------------------------------------------------------

Directional keys - Move the virus
ESCAPE key       - Exit The Virus

----------------------------------------------------------------------------------------------------------------
The Game
----------------------------------------------------------------------------------------------------------------

You are... a VIRUS! And your objective is to eat or kill all the cells in its host's organism.

But... it won't be easy, as the evil Defensins will try, with all their might, to destroy you.

The objective of the game is twofold: first, to survive for 5 minutes. The last, and more important, is
to kill all the cells in the game space.
